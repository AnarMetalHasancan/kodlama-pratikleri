# 100'lük not sisteminde kullanıcıdan girilen puan ile harf sonucu bastırma

Zorluk: Basit

Aşağıdaki harf notu - puan tablosunu kullanarak girilen puanı harf notuna çeviren bir algoritma geliştirmeniz bekleniyor. 

```
90-100   AA
80-89    BA
70-79    BB
65-69    CB
60-64    CC
50-59    DD
30-49    FD
0-29     FF
```

**Örnek:**

Input: 30
Output: FD

Input: 90
Output: AA

Input: 66
Output: CB
